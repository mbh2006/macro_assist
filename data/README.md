# Data Files

The textbook PDFs used during development are not included in this repository due to copyright restrictions.

To build the knowledge base:

1. Create a folder named `data/macro_textbooks`.
2. Place your legally obtained macroeconomics textbooks in that folder.
3. Run the ingestion pipeline.
4. The ChromaDB, BM25, and equation indexes will be generated automatically.

Generated indexes are excluded from version control.
